Image Converter Documentation

Welcome to the Image Converter documentation. This page provides information on how to use the Image Converter package for converting images to different formats.

The JPG to PNG Converter allows you to convert JPG images to PNG format. 

from image_converter import JPGtoPNGConverter

input_file_path = "input_image.jpg"
output_file_path = "output_image.png"

converter = JPGtoPNGConverter(input_file_path, output_file_path)
converter.convert()

The PNG to JPG Converter allows you to convert PNG images to JPG format. 

from image_converter import PNGtoJPGConverter

input_file_path = "input_image.png"
output_file_path = "output_image.jpg"

converter = JPGtoPNGConverter(input_file_path, output_file_path)
converter.convert()
